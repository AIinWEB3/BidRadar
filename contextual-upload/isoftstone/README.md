# iSoftStone Contextual Upload Pack

This folder contains test documents for the `bid-radar` MVP.

These files are:

- synthesized from publicly accessible iSoftStone materials
- intended for retrieval and demo testing
- not official bid documents or official representations from iSoftStone

Use them to populate a Contextual datastore so the skill can retrieve company-fit evidence during qualification.

## Recommended upload order

1. `01-company-overview-and-capabilities.md`
2. `02-consulting-delivery-and-industry-coverage.md`
3. `03-cloud-ai-and-data-capabilities.md`
4. `04-reference-projects-public-sector-and-education.md`
5. `05-certifications-and-partner-ecosystem.md`

## Suggested test questions

- What evidence suggests iSoftStone can deliver cloud and data modernization work?
- Does iSoftStone have public proof of transportation, education, or government-adjacent delivery?
- What certifications or capability signals support a bid on an IT services opportunity?
- What gaps exist if the opportunity requires deep U.S. federal past performance or classified work?

## Usage note

For realistic testing, these files intentionally include both strengths and gaps. That helps `bid-radar` return `Bid`, `Review`, or `No-Bid` instead of always leaning positive.
